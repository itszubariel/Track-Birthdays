Installation Instructions

To install this APK, you'll need to enable "Install from unknown sources" in your Android settings. This is completely safe — the app is open source and you can review the full source code on GitHub .

Why not on Play Store? To keep it 100% free with no tracking, ads, or data collection. Sideloading gives you complete control.
